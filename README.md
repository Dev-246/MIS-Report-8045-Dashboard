A comprehensive Management Information System (MIS) Dashboard built using Power BI to analyze business proposals, financing trends, and regional performance for decision-making.

---

## 📊 Visual Insights

### 1. Business Proposal by Occupation
- Government salaried customers dominate  
- Followed by self-employed and no-income categories  

### 2. Year-wise Approval Trend
- Peak approvals in 2020 (812)  
- Significant drop in 2021 (73) and 2022 (1)  

### 3. Application by House Type
- Majority applications for Houses (835)  
- Limited applications for Flats (58)  

### 4. Regional Distribution
- Highest contribution from:
  - Karachi  
  - Lahore  
  - Multan  
- Lowest from Sukkur  

### 5. Limit Type Analysis
- Most financing used for:
  - Construction of House  
  - Purchase  
- Minimal usage in:
  - Balance Transfer  
  - Flat Construction  

### 6. Proposal Status (Stage Code)
- Majority proposals are in DISB (Disbursed)  
- Few in other stages (RDISB, DISB4, etc.)
